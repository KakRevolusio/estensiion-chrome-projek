Kebijakan Privasi PWA Performance Checker

1. Informasi yang Dikumpulkan

Ekstensi PWA Performance Checker mengumpulkan data performa halaman web secara sementara selama proses pengujian, termasuk:

  • Metrik performa halaman web
  • Ukuran viewport
  • Status Service Worker
  • Waktu muat halaman

2. Penggunaan Data

Data yang dikumpulkan digunakan secara eksklusif untuk:

  • Menghitung performa website
  • Membandingkan versi PWA dan non-PWA
  • Menampilkan hasil analisis dalam ekstensi

3. Penyimpanan Data

Semua data diproses secara real-time dan tidak disimpan:

  • Tidak ada data yang disimpan setelah pengujian
  • Tidak ada data yang dikirim ke server eksternal
  • Seluruh proses analisis terjadi di browser pengguna

4. Privasi Pengguna

Kami berkomitmen melindungi privasi pengguna:

  • Tidak mengumpulkan data pribadi
  • Tidak melacak aktivitas browsing
  • Tidak menggunakan cookies atau penyimpanan persistens

5. Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui. Pengguna dianjurkan untuk secara berkala memeriksa halaman ini.
